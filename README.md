# li4OS

local-first p2p bench for digital mindfulness using blueberry

## technology stack
- react
- electron
- mobx
- p5
- hypercore / cabal
- blueberry

## install
$ npm install

## run
$ npm start

## use
0. set device name
1. set intention
2. host
3. send host key to peers
4. as peer, connect using key

## test
view console log for streamed data
